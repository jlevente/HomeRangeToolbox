Home Range Toolbox for ArcGIS Pro
=================================

**HomeRangeToolbox** is an ArcGIS Python Toolbox for spatial ecologists. It contains a number of tools to estimate home ranges and space utilization from point observations. The tools integrate into ArcGIS pro and are run using a graphical user interface, therefore they are suitable for GIS users. No coding skills are required.

The source code is available under the `MIT License <https://github.com/jlevente/HomeRangeToolbox/blob/main/LICENSE>`_.

To report issues, request features and download the tool, go to the `GitHub Repository <https://github.com/jlevente/HomeRangeToolbox>`_.

.. note::

   This project is under development.
